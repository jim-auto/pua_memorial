# Visual Novel Assets

Put generated PNG assets here with these exact filenames:

```text
bg-shibuya-night.png
man-neutral.png
woman-neutral.png
woman-good.png
woman-bad.png
```

The app automatically tries these files first. If they are missing, it falls back to the built-in SVG scene.

Recommended sizes:

```text
bg-shibuya-night.png  1920x1080
character PNGs       around 1400x2200, transparent background
```

Prompts are in `IMAGE_GENERATION_PROMPTS.md`.
